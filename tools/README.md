# tools
Инструменты для сборки SPA

# Node.js + Vite
[nodejs.org](https://nodejs.org/en/download)

```powershell
# winget install OpenJS.NodeJS

winget install Volta.Volta
volta install node@25
node -v
# v25.1.0
npm -v
# 11.6.2
```



[vite.dev](https://vite.dev/guide/)

```powershell
npm create vite@latest vite-vue-app -- --template vue
cd vite-vue-app
npm install
npm run dev
```



# Vuetify
Проверяем собираемость компонент от Google's Material Design.

[vuetifyjs.com](https://vuetifyjs.com/en/getting-started/installation/#using-vite)

```powershell
npm create vuetify@latest vite-material-vuetify
cd vite-material-vuetify
npm install
npm run dev
```
